

# Welcome to the Brain Games!

# Description
Ready for some brain games?
Five fascinating games await you. It won't be easy.

## Instructions
You need a console to play games.
Just type in the name of the game you want:

+ Brain-prime
+ Brain-even
+ Brain-progression
+ Brain-gcd
+ Brain-calc
